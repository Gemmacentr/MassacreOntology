Queries

Twelve competency questions have been identified with the intent to explore ontology and show its peculiarities.

Clicking on each square will appear the answer obtained through SPARQL, which is a query language for data represented through the Resource Description Framework (RDF).

SPARQL is one of the key elements of paradigm-related technologies known as semantic web, and allows you to extract information from knowledge bases distributed on the web.

The RDF language describes concepts and relationships on them through the introduction of triples (subject-predicate-object), and allows the construction of queries based on triple patterns, logical conjunctions, logical disjunctions, and optional patterns.
